# TankWar
可以自动寻路的坦克大战
a simple TankWar game developed with C++

### 技术栈
+ C++/C
+ EasyX
+ DFS深度优先搜索

### 运行环境/依赖
+ Windows 10
+ Visual Studio 2017
+ EasyX graphics library
